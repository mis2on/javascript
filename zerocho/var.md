# 변수\(var\)

var\(variable\) 변수를 만들어 컴퓨터한테 값을 기억하도록 할 수 있다.

```javascript
3 * 5
// 3 * 5 의 결과값 15를 결과라는 변수에 저장
var 결과 = 15
결과    //콘솔에 결과를 입력하 
15     //15 를 반환한다.

var 이름 = '몽실'
이름
'몽'

var 참거짓 = false
참거짓
false

var 결과 = 3 * 6
결과
18

```

## 변수명 네이밍 규칙

변수명에는 **$**와 **\_**를 제외한 특수문자는 사용할 수 없고, 숫자를 첫 글자로 사용할 수 없다.  
_보통 영문을 많이 사용  
 var name = '몽실'  
 var user\_name = '몽시리'_
