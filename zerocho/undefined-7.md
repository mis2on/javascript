# 배열

## 배열

값 그룹화는 하고 싶지만 속성 이름을 따로 주고 싶지는 않을 때 사용한다.

`var 배열 = [  
  '',  
  '',  
  ''  
}`

```javascript
var 배열 = [
  '사과',
  '오렌지',
  '포도',
  '딸기'
]


//결과값 
배열[0]
"사과"
배열[1]
"오렌지"
배열[2]
"포도"
배열[3]
"딸기"
배열[4]
undefined
배열[-1]
undefined
배열
(4) ["사과", "오렌지", "포도", "딸기"]
0: "사과"
1: "오렌지"
2: "포도"
3: "딸기"
length: 4
__proto__: Array(0)  //아직 몰라도됨
```
