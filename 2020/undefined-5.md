# 숫자야구

-화면구성

![](../.gitbook/assets/image%20%2839%29.png)





.join\(\) : 기본적으로 콤마\(,\)가 들어간다.  
.join\(''\) : 기본적으로 들어가는 콤마\(,\)를 없앨려면 빈문자열을 넣어준다.

버그  
4자리 숫자를 입력하면 앞에 0 자동삭제되어 네자리수가 되지 않는다.  
따라서 숫자가 아닌 문자로!

```javascript
Number('3046')
//3045
Number('0456')
//456
Number('0036')
//36
```

 최대한 적을수록 좋다.
